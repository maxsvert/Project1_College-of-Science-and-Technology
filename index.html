Our Majors

Web Development
First Major
Web development is the process of creating and maintaining websites or web applications. It involves two main aspects: front-end development, focusing on the user interface and user experience, and back-end development, dealing with server-side logic and databases. Developers use HTML, CSS, and JavaScript for front-end, along with various server-side languages and frameworks for back-end. Responsive design, security considerations, and the use of tools like code editors and version control are essential elements in web development. The goal is to build functional, visually appealing, and secure websites that cater to user needs.
Networking
Second Major
Networking involves connecting computers and devices to share information. It includes local networks (LANs), wide-area networks (WANs), and the internet. Key components include routers, switches, and protocols like TCP/IP. Wireless technologies like Wi-Fi and security measures such as firewalls and encryption play essential roles. Networking enables communication and data exchange, forming the foundation of modern computing and the internet.
Graphic Design
Third Major
Graphic design is the visual communication of ideas using images, typography, and layout. Graphic designers create visuals for various purposes, such as branding, advertising, and web design. They use elements like color, typography, and composition to convey messages effectively. Graphic design encompasses both print and digital media and plays a crucial role in establishing brand identity and enhancing user experience. Popular tools include Adobe Creative Suite, with software like Photoshop and Illustrator.
C# Programming
Fouth Major
C# (pronounced "C-sharp") is a versatile and modern programming language developed by Microsoft. It is part of the .NET framework and is widely used for building various types of applications, including desktop, web, and mobile applications. C# is known for its strong type-checking, object-oriented programming features, and memory management capabilities. It is commonly used in conjunction with Microsoft technologies and tools, and it's particularly prevalent in the development of Windows applications and backend services. C# is appreciated for its simplicity, readability, and robust support for building scalable and maintainable software solutions.
C++ Programming
Fitth Major
C++ is a versatile and powerful programming language that combines low-level features from C with object-oriented programming. Known for its efficiency and performance, it supports multiple programming paradigms, including procedural and generic programming. C++ is widely used in various applications such as game development, system programming, and embedded systems. It offers a standard template library (STL) for data structures and algorithms, and its code can be compiled to run on different platforms. Despite its steep learning curve, C++ is valued for its flexibility, control over system resources, and active community support.
